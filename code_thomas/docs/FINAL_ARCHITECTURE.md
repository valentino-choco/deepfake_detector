# Architecture finale

## Vue d'ensemble

Le projet conserve deux familles de signaux :

1. **Signaux de structure**
   - sections ;
   - ordre ;
   - champs attendus ;
   - alignement ;
   - espacement ;
   - polices.

2. **Signaux visuels**
   - incohérence d'en-tête / pied de page ;
   - zones graphiques ;
   - logos ;
   - densité d'encre ;
   - cohérence multi-pages.

Ces signaux sont fusionnés avec une couche de **revue ciblée par candidats numériques**.  
Le score final n'est pas un score de preuve absolue, c'est un score d'**authenticité opératoire** pour prioriser la revue humaine.
