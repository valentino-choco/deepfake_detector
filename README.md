# Deepfake Detector

A tool for detecting deepfake images and videos.

## Description

This project provides a deepfake detection system that can identify manipulated media content using machine learning techniques.

## Installation

```bash
git clone git@github.com:valentino-choco/deepfake_detector.git
cd deepfake_detector
pip install -r requirements.txt
```
